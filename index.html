TypeScript

It adds additional syntax to JavaScript to support a tighter integration with your editor, allowing you to catch errors early. TypeScript code converts to JavaScript, which runs anywhere JavaScript runs: in a browser, on Node.js, or Deno, and in your apps. Safety at scale is a key benefit of TypeScript..

What is TypeScript?

TypeScript is JavaScript with syntax for types. It provides a way to describe the shape of objects and functions in your code, making it possible to see documentation and issues in your editor TypeScript infers types based on runtime behavior. You can create and type JavaScript variables using TypeScript. It’s easy to provide types to JavaScript ES6 classes.
